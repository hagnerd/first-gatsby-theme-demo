## Checking out this demo

If you'd like to download this demo run

```sh
git clone git@github.com:hagnerd/first-gatsby-theme-demo
cd first-gastby-theme-demo
```

And using your preferred method either:

```sh
yarn install
```
or 
```sh
npm install
```
